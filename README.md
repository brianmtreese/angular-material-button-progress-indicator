# Angular Material Button Progress Indicator Demo
 
This repository demonstrates Angular Material's new built-in progress indicator support for buttons, introduced in Angular Material v22.
 
## What This Demo Shows
 
This app demonstrates a reports list where each row has a download button. It covers:
 
- Using the new `[showProgress]` input on `matButton` to control spinner visibility
- Projecting a `mat-progress-spinner` into the new `progressIndicator` content slot

## Running the App
 
```
npm install
npm start
```
 
Then open `http://localhost:4200`.
 
## Requirements
 
- Angular v22.0.0-next.0+
- Node.js 18+
